# Personal Portfolio

Personal portfolio website for Reeshav Raj, built with Next.js, React, Tailwind CSS, and Framer Motion.

## Highlights

- Responsive single-page portfolio with sections for about, skills, projects, experience, education, and contact
- Featured live projects including InternSphere, Automated-Data-Verrification, ResuMate, and the CSVTU Academic Portal
- Ready for GitHub hosting and Vercel deployment

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion

## Local Development

1. Install Node.js 22.
2. Install dependencies:

```bash
pnpm install
```

3. Start the development server:

```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000)

## Production Build

```bash
pnpm build
pnpm start
```

## Deploy On Vercel

1. Push this repository to GitHub.
2. Import the repository into Vercel.
3. Keep the detected framework as `Next.js`.
4. Use Node.js 22 in the project settings if Vercel asks for a runtime version.
5. Deploy.

Vercel will automatically use:

- Install command: `pnpm install`
- Build command: `pnpm build`
